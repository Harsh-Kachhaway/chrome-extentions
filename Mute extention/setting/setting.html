How to Use This Extension

Features

  • See a list of all your open tabs.
  • Click a tab name to switch to it.
  • Use the ❌ button to close any tab.
  • Right-click a tab entry to remove it quickly.
  • Mute/Unmute tabs individually or globally.

Tips

- Use the search bar to quickly find tabs by title or URL. - Toggle "Mute inactive tabs" if you want only your active tab to play sound.

Note: All changes apply instantly. You don’t need to reload Chrome.
